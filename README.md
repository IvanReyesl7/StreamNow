## Instalación

Iniciar Git en bash

```bash
  git init
```

Clonar Repositorio
```bash
  git clone https://github.com/IvanReyesl7/StreamNow.git
```
Instalar dependencias
```bash
  npm install
```

Iniciar proyecto
```bash
  cd StreamNow
  npx expo start
```
